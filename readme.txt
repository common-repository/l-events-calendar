== Installation ==

1. Upload `l-events-calendar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Drop the shortcode `[l-events-calendar]` into page, or you can click l-events button on top of your editor.
4. Click on the new admin menu item "L-Events" and create your event.
